Steam Deck · Game Mode Flatpak Apache-2.0 pre-alpha

An unofficial native client for YouTube's TV interface, built on Chromium-based Cobalt — controller-native, hardware-decoded, with clean sleep/resume.

Install on your Steam Deck

One-click (Desktop Mode) — installs the app and adds our repo so you get automatic updates:

⬇ Install Deckback

Opens in your software installer. Needs flatpak ≥ 1.16 (SteamOS 3.5+).

From a terminal (Konsole) — add the repo once, then install:

flatpak remote-add --if-not-exists deckback \
  https://properrr.github.io/deckback/deckback.flatpakrepo
flatpak install deckback io.github.properrr.deckback

Update anytime with flatpak update io.github.properrr.deckback — or let the Deck's background updater handle it.

Or a single bundle (no auto-update) from the GitHub Releases page:

flatpak install --user -y io.github.properrr.deckback.flatpak

Add it to your library

  1. Back in Game Mode: Add a Non-Steam Game → Deckback, or run steamos-add-to-steam (see the README).
  2. Open the shortcut's controller settings and apply the community Deckback layout.
  3. Controls: A select · B back · X details · Y search · LB/RB seek ∓10s.
